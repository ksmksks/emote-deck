EmoteDeck Privacy Policy

Last updated: 2026-03-04

This Privacy Policy explains how EmoteDeck ("the Extension") handles information when used on Twitch.

1. Data We Collect

EmoteDeck does not operate its own backend server and does not directly collect or store personal data on external systems.

The Extension uses Twitch-provided data and configuration features needed to render stamps and badges.

2. Data Sources Used in the Extension

  • Twitch Extension authorization context (for secure extension runtime operation)
  • Twitch Helix API responses required for panel display
  • Twitch Extension Configuration (broadcaster settings saved in Twitch)

3. How Data Is Used

  • Render channel stamps and badges in the extension panel
  • Apply broadcaster customization settings (layout/theme/visibility)
  • Provide panel interactions (for example, copy stamp text)

4. Data Sharing

EmoteDeck does not sell personal data and does not provide user personal data to third parties outside Twitch platform functionality.

5. Data Retention

Broadcaster settings are retained through Twitch Extension Configuration according to Twitch platform behavior. EmoteDeck does not retain separate copies on external servers.

6. Children's Privacy

EmoteDeck is provided through Twitch and is not directed to children under applicable minimum age requirements of the platform.

7. Changes to This Policy

This policy may be updated when extension behavior changes. The latest version will be published at this URL.

8. Contact
